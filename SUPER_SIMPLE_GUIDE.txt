STEP 1: Get Your Railway MySQL Password
────────────────────────────────────────────────────────────────

1. Open your browser
2. Go to: https://railway.app
3. Click on "MySQL" service (the database icon)
4. Click "Connect" tab
5. You'll see: MYSQL_PASSWORD=********
6. Click the "show" button next to ********
7. COPY the password (Ctrl+C)

────────────────────────────────────────────────────────────────

STEP 2: Run the Import Tool
────────────────────────────────────────────────────────────────

1. In File Explorer, go to: D:\management
2. Find file: RUN_THIS.bat
3. Double-click it
4. A black window will open
5. It will ask: "Paste your Railway MySQL password here:"
6. RIGHT-CLICK in the window and select "Paste"
   (Or just press Ctrl+V)
7. Press ENTER
8. Wait 10 seconds
9. You'll see: "🎉 IMPORT SUCCESSFUL! 🎉"

────────────────────────────────────────────────────────────────

STEP 3: Test Your Railway App
────────────────────────────────────────────────────────────────

1. Go to Railway dashboard
2. Click "Leave-management" service
3. Find your app URL (in Settings → Domains)
4. Visit: https://your-app-url.railway.app
5. Try logging in:
   Email: manager@company.com
   Password: Admin@123
6. Should work! ✅

────────────────────────────────────────────────────────────────

THAT'S IT! 🎉

Total time: 2 minutes

────────────────────────────────────────────────────────────────

TROUBLESHOOTING

Problem: "Access denied" error
→ Copy the password again from Railway
→ Make sure you clicked "show" button
→ Try again

Problem: "Connection refused"
→ Check MySQL service is "Online" (green dot) in Railway
→ Wait 1 minute and try again

Problem: RUN_THIS.bat doesn't work
→ Open PowerShell in D:\management folder
→ Type: node auto-import.js
→ Press Enter
→ Follow the prompts
